I am using run `5 (ayaexpanse8b)` as the basic level for all comparisons, because:
- I run it for all languages
- I used the default English prompt here
- Aya-Expanse-8B model is considered a generally good 8B model for 'LLM as a judge' use

Below are results for comparative runs (I mostly just numbered them):
10b, 11b, 11c, 11d - Aya-Expanse-8B with different CS and DE translations of different English prompts
11e, 12 -  - Aya-Expanse8B with English 7-hybrid scale prompts 
13 - Selene-1-Mini-Llama-3.1-8B with English 7-hybrid scale prompt
14 - Qwen3-8B with English 7-hybrid scale prompt
16 - Llama-4-Maverick with English 7-hybrid scale prompt

Note:
- in CSVs I use eg. 'aya-expanse-8b' or  'aya-expanse-8b_pcsdeh4' strings, ie. small caps
- entirely compacted 'ayaexpanse8b' strings are used only occasionally, eg. for filenames

Suffixes that I used:
_p - run makes something with the prompt(s)
_?csde - runs mostly concentrated with 'cs' and 'de' languages (it may still run for 'en' and 'ru' as a certain background)
_?????h5 - which 'judge_templates_locales_[hybrid][5].py' file I really used, it contains all prompt templates 
           eg. 'h' for 'hybrid' and '5' here, ie. 'judge_templates_locales_hybrid5.py' was used
